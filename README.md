# Picatown Mining

这是基于Asthenias的皮卡堂挖矿脚本的修改版本，感谢Asthenias的原创作品！

## 优化内容
1. 自动检测是否被炸出，并循环进入矿洞，在游戏初始界面直接启动程序即可。
2. 对矿场范围做更详细的界定，以减少落入矿场外的无效点击。
3. 限定相邻两次挖矿地点之间的距离，以免过长浪费行走时间。
4. 添加空格作为暂停/继续键。

## 使用须知
1. 请使用皮卡堂微端登录，运行程序时不要点开任何其他界面
2. 请勿将默认房间的背景设置成全黑，否则就无法检测出是否被炸出矿洞了～
3. 请勿隐藏工具栏和移动地图界面的初始位置
4. 按下ESC退出程序，按下空格暂停/继续程序
5. 一些关键数值的目前值：
   每40秒检测是否被炸出，每6秒点击“进入下一层”，每次等待角色走到挖矿位置的时间为0.5秒
   可以根据自己的幸运值和卡顿情况调整噢
